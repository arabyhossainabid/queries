এই ফাইল এ আমি কি কি ব্যবহার করেছি সেই SQL query গুলো সংক্ষেপে ব্যাখ্যা করা হয়েছে।

## Query 1

এই query ব্যবহার করে বুকিং সম্পর্কিত তথ্যের সাথে **কাস্টমারের নাম** এবং **ভেহিকেলের নাম** দেখানো হয়।

- INNER JOIN

## Query 2

এই query ব্যবহার করে এমন ভেহিকেলগুলো বের করা হয় যেগুলো কখনো বুক করা হয়নি।

- NOT EXISTS

## Query 3

এই query ব্যবহার করে নির্দিষ্ট টাইপের (যেমন car) এবং বর্তমানে **available** থাকা ভেহিকেলগুলো বের করা হয়।

- WHERE clause

## Query 4

এই query ব্যবহার করে প্রতিটি ভেহিকেল কতবার বুক হয়েছে তা গণনা করা হয় এবং যেগুলো একাধিকবার বুক হয়েছে শুধুমাত্র সেগুলো দেখানো হয়।

- GROUP BY
- HAVING
- COUNT
